## 游戏安装与食用指南
下载Release目录下的安装包，解压到同一目录，运行3个exe文件  
下载https://git.xeondev.com/WavyRooms/helios/releases，将这些文件放入游戏安装目录\Client\Binaries\Win64  
下载https://git.xeondev.com/RabbyDevs/zigrika-pakfile/releases，将这些文件放入游戏安装目录\Client\Content\Paks  
用管理员权限运行helios_launcher.exe  
启动游戏后，在左上角登录ip填http://127.0.0.1:10001  
  
自行构建可参考[该编译脚本](https://github.com/nyfair/wicked-waifus-buildbot/blob/master/.github/workflows/ww.yml)
